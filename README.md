# pumpkinpad
laser_ninja's Pumpkinpad firmware update

An update on the firmware to laser_ninja's Pumpkinpad, this enables extra key function compatibility (e.g., cut, copy, paste, media play, etc.) on the pad. 

Note that the NKRO option is set to false, if you intend to use multiple keys all at once, this flag should probably be set to true.

If you're looking to update the firmware but don't know where to do so, use QMK Toolbox. **I am NOT responsible for you bricking your pad, do so at your own risk.**
